Import cards from a file

Your cards live in the Google Sheet. To bulk-add a deck from a CSV file, open the Sheet and use its own importer:

  1. Tap Open my Sheet below — it opens the cards tab. Then File → Import → Upload your CSV.
  2. Import location: Append to current sheet adds to your deck (it appends to the cards tab you just landed on). Replace current sheet is a fresh start — it erases existing cards and progress.
  3. Set “Convert text to numbers, dates, and formulas” → No. This matters: otherwise Google mangles entries like 1/2 into dates.

Importing is a laptop/desktop task — the Sheets mobile app can’t do File → Import.
